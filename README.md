- Project Title: Inventory Management System (Python, PyQt5, PostgreSQL)

- Description:
 -Designed and implemented a full-featured desktop-based Inventory Management System to track products, categories, suppliers, and stock levels. Developed a responsive GUI using PyQt5 and integrated a PostgreSQL backend for persistent data storage and querying.

  Key Features:
- Added, updated, and deleted inventory items with real-time table refresh and search functionality.
- Designed UI with Qt Designer (.ui files) and loaded dynamically in Python using PyQt5.
- Used PostgreSQL to store item details, categories, suppliers, and stock levels with proper indexing and foreign key relationships.
- Implemented stock quantity validation and alert system for low stock items.
- Applied modular design with clear separation between UI logic, database operations, and business rules.
  
  Technologies Used:
- Python, PyQt5, Qt Designer, PostgreSQL, psycopg2
